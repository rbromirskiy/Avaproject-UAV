🛠 Основні характеристики (ТТХ)
Назва: "Крила України"

Тип: Планер із V-подібним хвостом та штовхаючим гвинтом

Злітна маса: 7.4 кг

Час у повітрі: 112–114 хвилин

Камера: Sony ILX-LR1 (61 Мп)

Система запуску: Рейкова катапульта (3 метри, прискорення ~4G)

🚀 Особливості конструкції
Матеріали: Кевларове покриття фюзеляжу для посадки "на пузо" та карбоновий внутрішній скелет.

Навігація: Використання RTK для сантиметрової точності координат центрів фотознімків.

Захист: Автоматизовані механічні шторки для захисту дорогої оптики під час приземлення.
