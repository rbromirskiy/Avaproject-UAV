## 🛠 Основні характеристики (ТТХ)
* [cite_start]**Назва:** "Крила України" [cite: 223]
* [cite_start]**Тип:** Планер із V-подібним хвостом та штовхаючим гвинтом [cite: 154, 232, 233]
* [cite_start]**Злітна маса:** 7.4 кг [cite: 117, 152]
* [cite_start]**Час у повітрі:** 112–114 хвилин [cite: 133, 167, 203]
* [cite_start]**Камера:** Sony ILX-LR1 (61 Мп) [cite: 24, 73, 170]
* [cite_start]**Система запуску:** Рейкова катапульта (3 метри, прискорення ~4G) [cite: 120, 123]

## 🚀 Особливості конструкції
1. [cite_start]**Матеріали:** Кевларове покриття фюзеляжу для посадки "на пузо" та карбоновий внутрішній скелет[cite: 101, 102, 103, 105, 115, 178].
2. [cite_start]**Навігація:** Використання RTK для сантиметрової точності координат центрів фотознімків[cite: 48, 242].
3. [cite_start]**Захист:** Автоматизовані механічні шторки для захисту дорогої оптики під час приземлення[cite: 75, 112].
